# Package Delivery Tracker

Overview:

This is a simple Python based Package Delivery Status Tracker. It helps in tracking your package/parcel ordered in terms of dispactched or in transit or out for delivery.

It uses:

CSV file storage

Modular Python package structure

CLI-based user interaction

CRUD operations

Status tracking with logs


Features:

Add a package with tracking ID, sender, receiver, and notes.

Update delivery status

View a single package

View all packages

Delete a package

CSV-based persistent storage

Timestamped activity log


Technologies/tools used:

Python: Core programming language

CSV file handling: Data Storage

VS Code: Development environment

Git and Github: Version control and hosting

Command line interface (CLI): User interaction

Standard Python libraries: csv,datetime,os



Steps to install and run the project:

Clone the repository

git clone <your-repo-link>
cd package_delivery_tracker_repo

Ensure Python is installed

python3 --version

Run the application

python3 -m package_delivery_tracker.cli


Instructions for testing:

Run the program

Choose 1: Add sample packages

Choose 4: Verify they appear

Choose 2: Update status of a package

Choose 3 → View details of an individual package

Choose 5 → Delete a package

Choose 4 → Confirm deletion

Choose 6 to exit
